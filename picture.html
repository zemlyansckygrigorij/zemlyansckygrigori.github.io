Работа с картинками


  1. Данная страница создана для учебных действий с картинками.
  2. При одиночном клике левой кнопкой мыши на центральную картинку она исчезает.
  3. При наведении мыши на центральную картинку у нее появляется рамка с цветом выбранным из выпадающего списка в левом верхнем углу.
  4. При прокручивании колесика мыши над центральной картинкой её размеры изменяются.
  5. При при одиночном клике левой кнопкой мыши на любую из верхних картинок она занимает место центральной.
  6. При наведении мыши на любую из верхних картинок у нее появляется красная граница.
  7. При нажатии мыши на любую из верхних картинок данную картинку можно переместить в любое место страницы.
  8. При нажатии кнопки "Добавить картинку" внизу добавляется картинка следующяя за данной (порядок картинок определяется при загрузке).
  9. При нажатии кнопки "Удалить картинку" нижняя картинка удаляется.
  10. При нажатии кнопки "Создать рамку картинки" у центральной картинки появляется рамка с цветом выбранным из выпадающего списка в левом верхнем углу.
  11. При нажатии кнопки "Сделать картинку фоновым изображением" центральная картинка становится фоновым рисунком .
  12. При нажатии кнопки « центральная картинка заменяется на предыдущюю (порядок картинок определяется при загрузке) .
  13. При нажатии кнопки » центральная картинка заменяется на следующюю (порядок картинок определяется при загрузке) .
